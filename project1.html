Software Engineering Project

Ideality

Ideality is a mobile app where users can go through catalogs of various different furniture, with a special feature where 3D replicas of the furniture can be placed in a simulation system, letting users plan out and create their ideal rooms.

App Preview

Tools and Technology Used

  • Figma (UI/UX Design)
  • Version Control: GitHub
  • Database: Firebase & Appwrite
  • Development IDE: Android Studio Koala Feature Drop 2024.1.2
  • AR Library: Google ARCore 1.46.0

Key Features

  • User account registration and login
  • Product search and detailed view
  • 3D virtual simulation for furniture preview
  • Shopping cart and order placement
  • Order history and cancellation
